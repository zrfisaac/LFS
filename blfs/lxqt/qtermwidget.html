qtermwidget-2.0.1

Introduction to qtermwidget

As the name suggests, the qtermwidget package is a terminal widget for Qt.

This package is known to build and work properly using an LFS 12.2 platform.

Package Information

  • Download (HTTP): https://github.com/lxqt/qtermwidget/releases/download/2.0.1/qtermwidget-2.0.1.tar.xz

  • Download MD5 sum: fe548f62dcc65064a84a54cbd133a0be

  • Download size: 188 KB

  • Estimated disk space required: 6.7 MB

  • Estimated build time: 0.2 SBU (Using parallelism=4)

qtermwidget Dependencies

Required

Qt-6.7.2

Installation of qtermwidget

Install qtermwidget by running the following commands: 

mkdir build &&
cd    build &&

cmake -D CMAKE_INSTALL_PREFIX=/usr \
      -D CMAKE_BUILD_TYPE=Release  \
      ..                           &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Program: None
Installed Libraries: libqtermwidget5.so
Installed Directories: /usr/include/qtermwidget5, /usr/lib/cmake/qtermwidget5, and /usr/share/qtermwidget5

Short Descriptions

libqtermwidget5.so

provides a terminal widget for Qt5
